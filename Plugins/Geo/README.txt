Provides geographical information from an ip address using GeoLite2.

requirements:

* GeoIP2-python

pip install geoip2

* GeoLite2 license key

1. Sign up for a MaxMind account (no purchase required) https://www.maxmind.com/en/geolite2/signup
2. Set your password and create a license key https://www.maxmind.com/en/accounts/current/license-key
3. Bot command: config plugins.Geo.licenseKey YOUR_LICENSE_KEY
